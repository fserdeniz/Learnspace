Write a program in C to sort elements of array in ascending order. 
## Test Data:
```c
Input the size of array : 5
Input 5 elements in the array :
element - 0 : 2
element - 1 : 7
element - 2 : 4
element - 3 : 5
element - 4 : 9
```

## Expected Output:
```c
Elements of array in sorted ascending order:
2 4 5 7 9
```
